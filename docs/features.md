# Features - Onyx Workspace

Onyx Workspace helps teams search private sources, ask grounded questions, run Deep Research, and package source-backed agent workflows with permission visibility.

## Permission-aware search

Find source-backed answers across internal systems while preserving the source boundary that made the answer trustworthy.

## Search, Ask, Deep Research, Agent

Use the right mode for exact source lookup, concise answers, longer synthesis, or reusable agent workflows.

## Connector planning

Map GitHub, Slack, Confluence, Google Drive, CRM, support queues, docs, and incident sources before rollout.

## Source visibility

Expose citations, freshness, source labels, and access assumptions so teams can verify the answer.

## Project bundles

Save approved source sets, instructions, and handoff notes for repeatable team workflows.

## Self-host and managed comparison

Help buyers decide when the open-source repo is enough and when a hosted operating layer adds value.

Plans focus on hosted workspace setup, connector planning, source visibility, team support, and operational rollout.
