# Guides - Onyx Workspace

Onyx Workspace helps teams search private sources, ask grounded questions, run Deep Research, and package source-backed agent workflows with permission visibility.

## Pick the first connector set

Start with the sources behind one real workflow instead of indexing every system at once.

## Review answer quality

Check citations, freshness, source ownership, permission fit, and whether the answer should become a saved project.

## Evaluate the GitHub repo

Inspect deployment paths, connector support, release activity, and whether your team prefers self-hosted ownership.

Plans focus on hosted workspace setup, connector planning, source visibility, team support, and operational rollout.
