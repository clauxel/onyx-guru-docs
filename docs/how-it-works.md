# How It Works - Onyx Workspace

Onyx Workspace helps teams search private sources, ask grounded questions, run Deep Research, and package source-backed agent workflows with permission visibility.

## Select source set

Choose the department, connector, project, or approved document bundle for the question.

## Run the right mode

Start with Search for exact sources, Ask for concise answers, Deep Research for synthesis, or Agent for reusable flow.

## Verify citations

Check which sources contributed, how fresh they are, and whether permissions match the user.

## Package the workflow

Turn recurring questions into saved projects, reports, or agent handoff notes.

Plans focus on hosted workspace setup, connector planning, source visibility, team support, and operational rollout.
