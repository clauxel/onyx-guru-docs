# Onyx Workspace Docs

Onyx Workspace helps teams search private sources, ask grounded questions, run Deep Research, and package source-backed agent workflows with permission visibility.

- Website: https://onyx.guru/
- Category: permission-aware private knowledge base, enterprise search, chat, Deep Research, and agent workspace
- Audience: teams connecting GitHub, Slack, Confluence, Google Drive, support tools, CRM, docs, and internal knowledge sources
- Support: support@aigeamy.com
- Upstream source context: [Onyx](https://github.com/onyx-dot-app/onyx)

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- A knowledge-base answer is only as good as the approved sources attached to the question.
- Connector permissions and sync timing must be tested before broad rollout.
- Sensitive actions should remain behind explicit human review and access controls.
- Onyx Workspace is an independent managed workflow around Onyx-style private knowledge search and does not claim upstream endorsement.
